"Hardware and Software"

Pengertian Hardware


Segala peralatan komputer yang dapat digunakan untuk mengumpulkan, memasukkan, memproses, menyimpan, dan mengeluarkan hasil pengolahan data dalam bentuk informasi.

Hardware terdiri dari:

  • Perangkat input
  • Perangkat Out put
  • Perangkat Proses
  • Perangkat Komunikasi

• Perangkat Input

Alat yang digunakan untuk memasukkan data ke dalam komputer


Misalnya: keyboard, mouse, digital camera, video camera, scanner, touch screen

• Perangkat Proses

Alat yang digunakan untuk memproses data


Terdiri dari: CPU dan Main Memory

• Perangkat Output

Peralatan yang digunakan untuk mengeluarkan informasi hasil dari pengolahan data


Misalnya: layar monitor, printer, LCD, speaker

• Perangkat Komunikasi

Peralatan yang digunakan untuk mengkomunikasikan data dari satu lokasi ke lokasi lainnya


Misalnya: network card, HUB, Fiber optic, modem, satelit, receiver

Pengertian Software


Kumpulan program yang digunakan untuk menjalankan aplikasi tertentu pada komputer.

Software terdiri dari:

  • System software
  • Application software

★ System Software

Mengendalikan dan mengoordinasikan operasi dari sistem komputer.

Dibagi menjadi 3 bagian:

  • Operating system
  • Utility
  • Language software

• Operating system

Mengendalikan dan mengoordinasikan operasi dari sistem komputer.


Misalnya: Apple DOS, Windows, UNIX


Fungsi OS:
  • Fungsi alokasi sumber daya
  • Fungsi penjadwalan sumber daya
  • Fungsi pengawasan terhadap aktivitas sistem komputer

• Language software

menerjemahkan instruksi-instruksi yang ditulis dalam bahasa pemrograman ke bahasa mesin sehingga bisa dimengerti komputer.


Misalnya: BASIC, PASCAL, C

★Application Software

Program yang ditujukan untuk menyelesaikan suatu permasalahan tertentu dalam aplikasi.

Dibagi menjadi 2:
  • General purpose
  • Spesific purpose
